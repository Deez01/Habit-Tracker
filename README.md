# StayonTrack

A habit tracking app with a train-themed interface. Complete habits to move your train along the track and maintain streaks.

## Features

- Create, edit, and delete habits
- Daily, weekly, monthly, one-time, and custom recurrence patterns
- Animated train progress on the home screen
- Current and all-time streak tracking
- Upcoming habits view with due dates
- Profile page with weekly/monthly/yearly completion charts
- Light and dark mode support
- Real-time updates via Supabase

## Tech Stack

- React Native (Expo)
- Expo Router (file-based navigation)
- Supabase (Authentication + PostgreSQL)
- TypeScript

## Prerequisites

- Node.js 18+
- npm or yarn
- Expo CLI
- iOS Simulator (Mac) or Android Emulator (or physical device with Expo Go)

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/stayontrack.git
   cd stayontrack
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the app:
   ```bash
   npx expo start
   ```
